MikHome — Starter
=================

Aperçu local (recommandé) :
1) Ouvrez ce dossier dans **VS Code**
2) Clic droit sur `index.html` → **Open with Live Server**
3) Naviguez sur http://localhost:5500

Changer les images :
- Les images actuelles pointent vers **Unsplash** (prévisualisation immédiate).
- Pour utiliser vos photos : remplacez les URLs `src` par des chemins locaux :
  /images/hero.jpg, /images/zone.jpg, /images/services/accueil.jpg, etc.
- Les fichiers et dossiers sont déjà créés (placeholders vides).
